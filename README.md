- ✅ **Drug-like** — QED ≥ 0.50, SA ≤ 4.0 (Ertl & Schuffenhauer), MW ≤ 500 Da
- ✅ **Synthesizable** — SA scores calculated using the industry-standard Ertl & Schuffenhauer algorithm

## Generation Method

- **Model:** 80M-parameter Causal Transformer with FlashAttention optimization
- **Architecture:** Custom StableTransformerLayer, D_MODEL=768, 12 heads, 12 layers
- **Training Data:** 1.8M molecules from MOSES + ZINC-250k
- **Sampling:** Top-P nucleus sampling (p=0.85, T=0.80)
- **SA Scoring:** Ertl & Schuffenhauer algorithm via RDKit sascorer
- **Novelty Filter:** RAM-indexed reference set of 4,643,595 unique SMILES from MOSES, ZINC-250k, and ChEMBL
- **Hardware:** NVIDIA RTX 5070 (Blackwell), fully local — no cloud compute
